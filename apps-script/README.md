Google Apps Script — Portfolio Data Entry
==========================================

Tentang Project Ini
-------------------
Script ini adalah contoh otomatisasi data entry menggunakan Google Apps Script
yang dibuat berdasarkan pengalaman nyata mengelola database personel security
di Kementerian Perdagangan RI (2021-2026).

Fitur:
1. Auto-populate dropdown Google Form dari Google Sheets
2. Export database ke CSV otomatis
3. Kirim email reminder pengisian form

Cara Penggunaan:
1. Buka Google Sheets → Extensions → Apps Script
2. Copy paste file .gs
3. Ganti SHEET_ID, FORM_ID dengan milik sendiri
4. Set trigger (Edit → Current project triggers)

Dokumentasi Lengkap:
https://developers.google.com/apps-script

# Strategi Cari Kerja — Data Entry / Admin Remote

---

## 1. Target Posisi (Urut Prioritas)

| Posisi | Platform | Gaji Rate Remote |
|--------|----------|-----------------|
| **Data Entry Specialist** | LinkedIn, Upwork, Freelancer | $3-7/jam |
| **Administration Assistant** | LinkedIn, Glints, Indeed | $4-8/jam |
| **Virtual Assistant** | Upwork, OnlineJobs.ph | $3-6/jam |
| **Database Administrator** | LinkedIn, Jobstreet | $5-10/jam |
| **HR Admin / Payroll Support** | LinkedIn, Glints | $4-7/jam |

## 2. Platform Terbaik

| Platform | Tips |
|----------|------|
| **LinkedIn** | Update headline + aktif posting |
| **Upwork** | Buat profile lengkap, lamar 5 job/hari |
| **OnlineJobs.ph** | Fokus ke employer luar negeri |
| **Glints / Jobstreet** | Posisi lokal tapi bisa remote |
| **Freelancer.com** | Cocok untuk project-based |

## 3. Jadwal Apply (HARIAN)

- 09.00-10.00: Apply 5-10 job di LinkedIn & Glints
- 10.00-11.00: Apply 5 job di Upwork (kirim proposal custom)
- 11.00-12.00: Apply 5 job di OnlineJobs.ph
- Siang: Cek email, follow up lamaran

## 5. Target (30 Hari)

| Minggu | Target |
|--------|--------|
| **Minggu 1** | Apply 50+ lamaran, follow up |
| **Minggu 2** | Dapat 3-5 panggilan interview/tes |
| **Minggu 3** | Tes Data Entry dari perusahaan |
| **Minggu 4** | Offer / Join project |

## 6. Kunci Sukses

1. **Konsisten apply setiap hari** — ini yang paling penting
2. **Custom cover letter** — jangan copypaste
3. **Tunjukkan portfolio** — link portfolio website
4. **Kecepatan entry data** — di tes biasanya dinilai
5. **Akurasi** — lebih penting dari kecepatan
